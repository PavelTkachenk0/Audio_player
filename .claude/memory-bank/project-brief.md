# Project Brief

**Audio Player** — backend музыкального стримингового сервиса. Учебный/pet-проект, автор — один разработчик (изначально вырос из эксперимента с SignalR).

REST API: каталог (треки / альбомы / исполнители / жанры / плейлисты), избранное, рекомендации, сквозной поиск, аутентификация JWT + 2FA, потоковая передача MP3 через SignalR.

**Цель:** чистый, безопасный, презентабельный backend (для портфолио/резюме). Возможное развитие — учебная миграция на микросервисную архитектуру (особый интерес — брокеры сообщений на разрезе Catalog ↔ Library).
